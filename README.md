## 📖 소개 

개발자 중심으로 설계하고 있는 서비스 입니다.<br>
개발자의 불편함을 해소하고 정보 공유를 편하게 하는것을 목적으로
개발자의 편의성을 고려한 블로그 서비스 입니다.

---

## ✨ 주요 기능

- ✅ 마크다운 문법지원
- ✅ AI목차 요약

---

## 🛠️ 향후 업데이트할 기능

| 기능 | 설명 |
|------|-----------|
| **백업기능** | 서비스의 지속성 문제를 해결하고자 백업기능을 지원할예정입니다. |
| **이력서 템플릿 통합** | 개발자에게는 블로그가 이력서의 일부분이라고 생각합니다그렇기에 블로그에 이력서 템플릿을 함께 지원할 예정입니다. |
| **채용시장과 통합** | 기업에서 개발자의 블로그를 보고, 면접제안을 할수 있다고 생각합니다. 그렇기에 채용공고를 함께 지원할 예정입니다. |
| **종합된 기업 기술블로그** | 회사에서 작성한 기술블로그는 시행착오를 겪은 글들이 많이있습니다.  이러한 글들은 개발자에게 큰도움이 되므로 기업의 블로링크를 한번에 보여줄 예정입니다. |
| **AI기능 서포트** | 개발자가 글을 작성할때에는 근거자료와 참고자료가 필요합니다.  작성하는 동안에 AI를 통해 관련문서 링크, 관련글을 추천할 예정입니다. |
